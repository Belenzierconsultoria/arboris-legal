## 2. INFORMAÇÕES QUE COLETAMOS

### 2.1. Dados Fornecidos por Você

O Arboris coleta as seguintes informações que você fornece voluntariamente:

#### A) Dados de Projetos Florestais:
- Nome do projeto
- Descrição do projeto
- Nome do cliente/proprietário
- Nome do responsável técnico
- Município/localização (texto)

#### B) Dados de Levantamento Florestal:
- Informações de espécies arbóreas (nome científico, nome comum, família botânica)
- Medidas dendrométricas (CAP, DAP, altura)
- Número de indivíduos registrados
- Observações de campo (texto livre)
- Condição dos indivíduos (viva/morta)

#### C) Dados de Localização (GPS):
- Coordenadas geográficas (latitude e longitude) dos pontos de medição
- Coordenadas das áreas de estudo
- **Importante:** Coletamos coordenadas GPS APENAS quando você registra manualmente um indivíduo ou área. O app NÃO rastreia sua localização em tempo real.

#### D) Fotos:
- Fotos dos indivíduos arbóreos capturadas através do aplicativo
- As fotos são armazenadas localmente no seu dispositivo

### 2.2. Dados Coletados Automaticamente

#### A) Informações do Dispositivo:
- Modelo do dispositivo
- Versão do sistema operacional (Android/iOS)
- Versão do aplicativo
- Identificador único do dispositivo (para rastreamento de instalações)

#### B) Dados de Uso (Analytics):
- Recursos utilizados no aplicativo
- Tempo de uso do aplicativo
- Eventos de interação (criação de projetos, registro de indivíduos, geração de relatórios)
- **Importante:** Coletamos dados de uso de forma ANÔNIMA, sem identificar você pessoalmente

#### C) Dados de Erro (Crash Reports):
- Logs de erro quando o aplicativo apresenta falhas
- Informações técnicas sobre o erro para correção de bugs

### 2.3. Dados que NÃO Coletamos

O Arboris **NÃO coleta**:
- ❌ Nome completo, CPF, RG ou outros documentos pessoais
- ❌ Endereço residencial ou comercial completo
- ❌ Número de telefone
- ❌ Endereço de e-mail (a menos que você entre em contato com suporte)
- ❌ Informações bancárias ou financeiras (pagamentos são processados pela Google Play)
- ❌ Dados sensíveis (origem racial, convicções religiosas, saúde, etc.)
- ❌ Rastreamento contínuo de localização GPS

## 4. ARMAZENAMENTO E SEGURANÇA DOS DADOS

### 4.1. Onde Seus Dados São Armazenados

**IMPORTANTE:** O Arboris é um aplicativo **100% OFFLINE** que armazena todos os seus dados **LOCALMENTE** no seu dispositivo móvel.

#### Armazenamento Local:
- ✅ Todos os projetos, indivíduos, fotos e dados são salvos no banco de dados local do dispositivo (SQLite)
- ✅ Os dados permanecem apenas no seu dispositivo
- ✅ **NÃO** enviamos seus dados para servidores externos ou nuvem
- ✅ Você tem controle total sobre seus dados

#### Backup (Futuro - Opcional):
- Em versões futuras, poderemos oferecer backup em nuvem OPCIONAL
- Será necessário consentimento explícito antes de ativar
- Você poderá escolher entre Google Drive, Dropbox ou outros serviços

### 4.2. Segurança dos Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados:

#### A) Medidas Técnicas:
- Criptografia do banco de dados local (SQLite)
- Acesso restrito aos dados do aplicativo
- Atualizações regulares de segurança
- Proteção contra acesso não autorizado

#### B) Medidas Organizacionais:
- Políticas internas de proteção de dados
- Treinamento da equipe sobre privacidade
- Monitoramento de segurança

#### C) Limitações:
- Apesar de nossos esforços, nenhum sistema é 100% seguro
- Você é responsável pela segurança do seu dispositivo (senha, bloqueio de tela)
- Recomendamos fazer backups regulares dos seus dados

### 4.3. Período de Retenção

- **Dados locais:** Permanecem no seu dispositivo até você excluir manualmente (deletar projeto ou desinstalar app)
- **Dados de analytics:** Retidos por até 24 meses de forma anônima
- **Dados de suporte:** E-mails de suporte retidos por até 12 meses

#### D) Portabilidade:
